# Pavlovskis__UnityUI
Darbs ar UI elementiem Unity dzini

Unity 2D projekts, izstrādāts mācību ieskaites uzdevuma ietvaros.
Projektā tiek izveidota tēla apģērbšanas un aprīkojuma izvēles sistēma,
izmantojot Unity UI elementus un C# skriptus.

## Projekta mērķis
Izstrādāt lietotni, kurā lietotājs var:
- izvēlēties tēlu,
- ievadīt tēla vārdu un dzimšanas gadu,
- automātiski aprēķināt tēla vecumu,
- izvēlēties un uzlikt apģērbu un aprīkojumu,
- pārvietot objektus ar drag & drop principu,
- mainīt tēla izmērus ar slider elementiem,
- pārvietoties starp ainām.

## Izmantotās tehnoloģijas
- Unity 2D (Unity Hub 3.16.2)
- C#
- Unity UI sistēma
- Git versiju kontrole
- GitHub
